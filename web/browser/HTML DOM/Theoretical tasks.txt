Define the Document Object Model (DOM) and explain how it represents an HTML document.
List and describe the primary node types in the DOM, including their roles in the HTML structure.
Explain how the DOM differs from the original HTML structure of a page.

1. DOM это объектная модель html документа создаваемая браузером для управления структурой, содержанием или стилями страницы.
Это дерево объектов, в котором каждый узел представляет собой элемент, атрибуты, контент или комментарий.

2. DOM может содержать следующие узлы:
    ~ узлы элементов - представляют HTML элементы (теги)
    ~ узлы аттрибутов - представляют атрибуты элементов: стили, классы, ссылки, id
    ~ узлы текста - представляют текстовое содержимое элементов, например текст внутри тега <p>
    ~ узлы комментариев - представляют комментарии в коде HTML документа

3. HTML документ по своей сути является просто текстовым документом,который не может динамически изменяться.
 DOM же в свою очередь является древовидной структурой данных, используемой для изменения и взаимодействия "на лету" с содержимым страницы.
